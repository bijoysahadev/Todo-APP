# 📋 TaskMaster: CRUD Todo Application

A lightweight, high-performance **Todo App** built to showcase fundamental frontend engineering skills. This application allows users to manage their daily tasks efficiently with a smooth, responsive interface.

---

## ⚡ Key Features

* **Full CRUD Functionality**: Seamlessly Create, Read, Update, and Delete tasks.
* **Data Persistence**: Uses `localStorage` to save your tasks locally in the browser.
* **Real-time Updates**: Instant DOM manipulation without page reloads.
* **Responsive UI**: Optimized for all screen sizes (Mobile, Tablet, and Desktop).
* **Task Status**: Toggle tasks between "Pending" and "Completed."
